
# Media Type

Specifies type information for a media asset.

## Structure

`MediaType`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `image_type` | `string` | Required | Specifies an image format, as applicable. |
| `kind` | `string` | Optional | Specifies the broad "kind" of media. |

## Example (as JSON)

```json
{
  "imageType": "JPG"
}
```
